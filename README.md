# Wizz NutriMarket

Minimalist landing page oriented to the health and nutrition market.

Built for a well-known health and nutrition company in Salta, Argentina (Wizz NutriMarket).

### Technologies used

- HTML5
- SCSS/CSS
- JavaScript
